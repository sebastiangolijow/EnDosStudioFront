# StickerApp Frontend

Vue 3 SPA for a Barcelona-based print shop selling custom stickers. Customers upload a design, the editor auto-detects the cut line via OpenCV.js, they paint relief zones, preview the result, and place an order. Pairs with the Django backend at [EnDosStudioApp](https://github.com/sebastiangolijow/EnDosStudioApp).

Built by **YeKo Studio**.

**Stack**: Vue 3 · Vite · TypeScript · Pinia · Vue Router · Tailwind CSS · Axios · Canvas API · OpenCV.js (CDN).

For agent context (mindset, conventions, design tokens, API contract, gotchas): see [`CLAUDE.md`](./CLAUDE.md).
